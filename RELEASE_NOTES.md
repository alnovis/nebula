# Release Notes

## v0.2.21

**Release Date:** 2026-01-27

### Highlights

This release focuses on content enrichment and visual consistency. Project pages now match the styling of blog posts, and project cards on the homepage display technology tags for quick identification.

### New Features

- **Tags on project cards** — Up to 3 technology tags now appear next to the status badge on homepage project cards, helping visitors quickly identify relevant projects
- **Project detail page improvements** — Cover images and share buttons now appear on project pages, matching blog post styling

### Improvements

- **Expanded documentation content:**
  - "Redesigning My Blog" article expanded from ~120 to ~290 lines with sections on Critical CSS, CLS optimization, CDN fallback for Russia, and Mermaid lazy loading
  - Nebula project page expanded from ~200 to ~365 lines with Performance Optimizations, SEO & Social, Security, Development Workflow, and Infrastructure sections
  - Proto Wrapper Plugin page enhanced with Field Contracts documentation and Renumbered Fields Support section

### Bug Fixes

- Fixed Mermaid diagrams not rendering (missing `run()` call after initialization)
- Fixed Mermaid diagram centering and sizing in dark theme

### Migration

No breaking changes.
